SDWebImage源码阅读笔记

发表于 2017-08-21 | 分类于 iOS

说明

  1. SDWebImage地址: https://github.com/rs/SDWebImage
  2. 版本: 4.1.0
  3. 只分析 iOS OC 版本的相关实现, MacOS, watchOS等平台以及 Swift 版本暂不涉及, 代码实现大同小异

    项目目录结构及其功能

  4. SDWebImageOperation : 是一个protocol, 就定义了一个cancle方法; SDWebImageCompat 类: 主要是一些不同系统平台下的常用宏定义
  5. Downloader : 下载类,主要是对NSOperation以及NSURLSessionTask以及相关操作
  6. Cache : 缓存类, 主要是SDImageCache以及SDImageCacheConfig,内部涉及memory cache 和 disk cache的处理
  7. Utils : 工具类, 主要是管理调用上层接口,以及一些解压图片的工具,外加一个图片预取工具类
  8. Categories : 项目中使用系统类的扩展方法
  9. WebCache Categories : 一些 UI 控件的扩展方法,一般是我们直接调用的接口
